Windchill Document Worker

Automatically Generate PDFs of Office Documents Checked into Windchill

Download this project as a .zip file Download this project as a tar.gz file

Project Summary

The WindchillDoc2PDF script is a PowerShell script that automatically generates PDFs of Office documents as they are checked into Windchill. The PDFs then get stored as representations in Windchill for later viewing.
